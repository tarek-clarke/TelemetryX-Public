# Contributing to TelemetryX

Welcome! While TelemetryX is closed-source, we highly value community feedback, bug reports, and feature suggestions. This repository is the hub for all community interactions.

## How to Contribute

### 1. Reporting Bugs
- Check existing issues before opening a new one.
- Use the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md).
- Include system specs (OS version, GPU, Driver version).

### 2. Suggesting Features
- We love new ideas!
- Use the [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md).
- Describe the use case and how it benefits the community.

### 3. Reporting Performance Issues
- If the overlay causes framedrops or high CPU usage, let us know.
- Use the [Performance Issue Template](.github/ISSUE_TEMPLATE/performance_issue.md).

## Pull Requests
As this is a documentation-only repository, pull requests are restricted to:
- Improvements to documentation clarity.
- Fixing typos.
- Updating supported platform lists.

Implementation PRs for the core engine are not accepted in this repository.

## Community Standards
Please be respectful and constructive in all interactions. We follow the standard Contributor Covenant Code of Conduct.
